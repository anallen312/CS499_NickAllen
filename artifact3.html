Overview

This enhancement focuses on improving the software design and engineering principles within my Android Studio project.

Original Implementation

Initially, the project had issues with modularity, maintainability, and scalability. The following areas needed improvement:

  • Separation of concerns (MVC or MVVM pattern)
  • Better class structuring
  • Code reusability & refactoring

Enhancements Made

The following modifications were implemented to enhance software design:

        // Example: Before Refactoring
        public class MainActivity extends AppCompatActivity {
            private DatabaseHelper dbHelper;
            private Button btn;
            @Override
            protected void onCreate(Bundle savedInstanceState) {
                super.onCreate(savedInstanceState);
                setContentView(R.layout.activity_main);
                dbHelper = new DatabaseHelper(this);
                btn = findViewById(R.id.button);
                btn.setOnClickListener(view -> dbHelper.insertData("Sample"));
            }
        }
        
        // After Refactoring: Using MVVM
        public class MainViewModel extends ViewModel {
            private DatabaseRepository repository;
            public MainViewModel() {
                repository = new DatabaseRepository();
            }
            public void insertData(String data) {
                repository.insertData(data);
            }
        }

Impact of the Enhancement

By implementing the MVVM pattern, the project now follows better design principles, making the codebase more maintainable and scalable.

Conclusion

This enhancement significantly improved the software design quality of the project, aligning with industry best practices.
